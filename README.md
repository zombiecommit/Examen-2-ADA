## Ejercicios desarrollados
### Tree: Height of a Binary Tree
Ejercicio desarrollado en C++ que calcula la altura de un árbol binario mediante una solución recursiva.

### QHEAP1
Ejercicio desarrollado en Java que implementa las operaciones de inserción, eliminación y consulta del valor mínimo de un Min Heap.

## Ejecución de los programas
Para ejecutar las soluciones se debe ingresar a la carpeta correspondiente a cada ejercicio.

### Tree: Height of a Binary Tree
Requiere un compilador compatible con C++.

### QHEAP1
Requiere tener instalado el JDK de Java.

Los códigos también pueden ser ejecutados directamente en la plataforma HackerRank.
